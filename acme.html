Acme

June 2020

Stencil js, Sass

Description

Project developed during the Mohole School Digital Techniques course. The project was developed with Stencil Js with a particular focus on the use of web components.

I worked in a team of 3 people using Git to deploy the code.



Brief

Acme is a personal and business services company that includes financial and insurance services It is a company that has been present for some time but has decided to renew itself by proposing a Landing Page able to attract a younger audience.

I worked in a team of 3 people using Git to deploy the code.


- It will be necessary to emphasize the company's core business (financial and insurance services) and its target customers (companies and individuals).

- You will need to use the style rules defined in the styles file and the images provided by the company

Design

We decided to use different colored cards to divide the company's core business (financial and insurance services) and its target customers (companies and individuals).



Develop

Developing the site was pretty easy. We worked in a team of 3 on the various components without worrying about css class collisions since in Stencil Js each component is separate from the others.
